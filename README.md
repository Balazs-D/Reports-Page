### 1. Install

- Clone or download the repository

```javascript
git clone git@github.com:Balazs-D/elucidate-challenge.git
```

- Get into the directory

```javascript
cd elucidate-challenge
```

Install dependencies using

```javascript
npm install
```

- Run the application

```javascript
cd elucidate-challenge
npm start
```

### 2. Used Technologies

- React.js
- Styled Components
- Axios
